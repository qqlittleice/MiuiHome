# MiuiHome

__Hook for MIUI Launcher - Xposed__
 
### 自定义你的 MiuiHome

 __涉及：__

- 平滑动画.
- 文件夹模糊.
- 动画速度调节.
- 后台模糊级别.
- 状态栏时钟常显.
- 后台卡片圆角大小调节.
- 下载时图标的水波纹特效.
- 后台卡片图标文字间距调节.
- 强制为高端设备(默认启用).

### 激活模块后重启系统桌面,再打开桌面设置即可看见模块设置弹窗.
