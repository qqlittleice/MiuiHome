# MiuiHome  [![Android CI](https://github.com/1767523953/MiuiHome/actions/workflows/main.yml/badge.svg)](https://github.com/1767523953/MiuiHome/actions/workflows/main.yml)

 MiuiHome自定义-Xposed
 
### 允许用户自行选择模糊级别及特效

 __涉及：__
 
- 平滑动画.
- 动画速度.
- 文件夹模糊.
- 水波纹特效.
- 后台模糊级别.
- 状态栏时钟常显.
- 强制设备为高端设备.
